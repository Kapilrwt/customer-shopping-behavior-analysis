# 👨🏻‍💻 Customer Shopping Behavior Analysis

## 📌 Project Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, PostgreSQL, SQL, and Power BI. The objective was to analyze customer shopping behavior, identify purchasing trends, and generate actionable business insights through interactive visualizations.

## 🛠️ Tools & Technologies

- Python
- Pandas
- PostgreSQL
- SQL
- Power BI
- Jupyter Notebook

## 📊 Project Workflow

1. Data Collection and Import
2. Data Cleaning and Preprocessing using Python
3. Exploratory Data Analysis (EDA)
4. Loading Data into PostgreSQL Database
5. SQL-Based Business Analysis
6. Interactive Dashboard Development in Power BI
7. Business Insights and Recommendations

## 🔍 Key Analysis Performed

- Customer Segmentation Analysis
- Category-wise Sales Analysis
- Revenue Analysis by Age Group
- Subscription Status Analysis
- Customer Purchase Behavior Analysis
- Customer Review Analysis

## 📈 Dashboard Highlights

- Number of Customers
- Average Purchase Amount
- Average Review Rating
- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Sales by Age Group
- Interactive Filters

## 💡 Key Insights

- Clothing category generated the highest revenue.
- Adult customers contributed the highest sales volume.
- Average purchase amount was approximately $60.
- Customer purchasing behavior varied across categories and age groups.
- Subscription status influenced customer purchasing trends.
